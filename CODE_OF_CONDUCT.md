# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

All participants are expected to uphold it. Report concerns by opening a private
[GitHub Security Advisory](https://github.com/menemy/sideshell/security/advisories/new)
or by contacting a maintainer through GitHub.
